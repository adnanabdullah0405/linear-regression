##  Overview

This project implements a **Simple Linear Regression** model to predict a person's score based on their income. It is an introductory machine learning project that demonstrates essential steps such as data visualization, preprocessing, training, and model evaluation.

This project is suitable for beginners aiming to understand how regression works in supervised learning scenarios using Python's `scikit-learn`.
